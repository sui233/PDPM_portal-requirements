# 首页设计
首页是用户在访问网站、进行数据分析的过程当中所见到的第一个页面。这个页面对于用户的使用体验会有相当关键的影响。参考cBioPortal，PDPM portal的首页内容将主要包含“数据库内容总览”、“搜索入口”与“数据集浏览与选取”等主要组件，同时可以考虑添加“公告板”、友情链接等其他各类组件。首页的大体布局如下图所示：

## 数据库内容总览
这个部分参考GDC Data portal设计，对应图中的“Database Overview”部分。这个部分包括“浏览导航”与“数据库版本总结概览”两个部分。前者包含肿瘤的分类树，点击该分类树可以展开或收起之，点击某个节点时可以同时点选与该节点相关的所有节点；后者展示当前版本的数据库当中各类型的数据总体上有哪些，不需要交互功能，只需要展示功能，大体上需要展示：研究项目数量、疾病种类数、收录患者数、蛋白质种类数、Proteoform总数以及相应研究文章的数目。

## 搜索入口
主要用于搜索蛋白质，在未选取数据集时只能进行全局搜索，在选取数据集后可以在选区范围内进行检索。蛋白质的检索结果在“Custom Selection”的界面当中展示，即勾选用户搜索的蛋白质，并将其相应条目置顶。

这个搜索框也可以用于搜索其他与数据集本身相关的内容，例如病灶或疾病名称等，其搜索结果以如下方式体现：左边的癌症分类树的相关节点高亮，下方的数据集列表当中相应的条目被勾选。

## 数据集浏览与选取
该部分为类似cBioPortal主页部分的数据集列表的一个列表，每个条目对应一个研究项目，每个条目的左边有复选框，右边有“查看相应文献”、“内容速览（Express Overview）”等选项。用户可以选择一个或多个研究项目进行速览，或选择两个及以上的项目进行对比分析，这两种分析都可以在数据分析页面当中找到。
